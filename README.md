# ddanjit-kit — 딴짓연구소 수업 키트

딴짓연구소 클로드코드 강의 수강생용 plugin 창고(marketplace)입니다.

## 학생 설치 (수업 중 안내)

```
/plugin marketplace add ddanjitlab/ddanjit-kit
/plugin install ddanjit-class@ddanjit-kit
/ddanjit-class:practice
```

- 2번째 줄 하나로 기획 기능(**superpowers**)까지 함께 설치됩니다.
- 3번째 줄이 폴더정리 실습파일을 지금 열려 있는 폴더에 준비합니다.
- ⚠️ 설치 직후 `/ddanjit-class:practice`가 안 보이면 `/reload-plugins`를 먼저 실행하세요.

## 담긴 것

| plugin | 무엇 | 출처·라이선스 |
|---|---|---|
| **ddanjit-class** | 딴짓 수업키트 (실습파일 command + superpowers 번들) | 딴짓연구소 제작 |
| **superpowers** | 브레인스토밍·계획 등 기획 기능 | [obra/superpowers](https://github.com/obra/superpowers) · MIT (Jesse Vincent). 우리가 복제하지 않고 원본을 가리킵니다. |
